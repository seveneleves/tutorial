Tutoriel de vote

Etape 1

Installation de MetaMask Mobile

MetaMask est une application mobile qui vous permet de communiquer avec la blockchain Ethereum pour l'envoi et la réception de tokens.

Cliquez ici pour installer MetaMask mobile !

La création de votre portefeuille ne prend que quelques secondes.

Etape 2

Configuration du Token MC2I

Le TMC2I est un token personnalisé, c'est la raison pour laquelle il n'apparaîtra pas dans votre portefeuille MetaMask par défaut. Pour ce faire :

→ Dans MetaMask, cliquez sur l'icône du menu située en haut à gauche de la fenêtre

→ Cliquez sur "Wallet" puis, en haut de la page qui vient de s'ouvrir, sur "Ethereum Main Network"

→ Dans la liste des réseaux, sélectionnez "Ropsten Test Network"

→ Dans la fenêtre de votre portefeuille, cliquez sur "+ ADD TOKENS" puis sur l'onglet "CUSTOM TOKEN"

→ Dans le champ "Token Address", copiez-collez l'adresse suivante :

0xbf0776118b8c7e41b2409295307d557ca16fa989

→ Le reste des champs seront complétés automatiquement. Puis cliquez sur "ADD TOKEN". Le TMC2I apparaît désormais dans votre liste de tokens !

Etape 3

Récupération de l'adressse

Vous utiliserez MetaMask pour exprimer votre vote. Pour ce faire, vous disposez de deux éléments :

→ 0,1 ETH qui vous serviront à régler les "frais" fictifs de transaction

→ 1 TMC2I soit l'équivalent d'une voix au moment du vote

Etape 4

Votez !

Dans le champ "Amount", cliquez sur le triangle dirigé vers le bas, sélectionnez "TMC2I" puis cliquez sur "Max".

Laissez les autres paramètres inchangés puis cliquez sur "Next" et "Confirm". Félicitations ! Votre TMC2I a été envoyé à l'adresse indiquée.
